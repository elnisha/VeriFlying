## Relevant information:
__Large file management in github__ for the dataset, using `git lfs`:
To setup:

1. Install the git lfs
    - On linux `sudo pacman -S git-lfs`
    - On Windows: Download and run the installer from git-lfs.github.com
2. Run once per user: `git lfs install`
3. Track desired files with: `git lfs track "data/*"`, this will create a file named .gitattributes, that __must be comitted__.
